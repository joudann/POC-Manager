# 🛡️ POC Asset Manager (v1.0)

**POC Asset Manager** 是一款基于 Python 和 **CustomTkinter** 开发的轻量级漏洞验证脚本（POC）管理工具。它采用现代化的扁平风格 UI（暗色模式），帮助安全研究人员高效地管理、检索和同步本地的 POC 脚本库。

> **注意**: 此版本 (v1.0) 使用 `CustomTkinter` 框架，与使用 PyQt6 的 v2.0 版本在底层架构上有所不同。

## ✨ 主要功能

- **🎨 现代 UI**: 默认采用深色模式 (Dark Mode)，界面简洁美观。
- **🔍 快速检索**: 基于 SQLite 本地索引，支持按关键字（如 CVE 编号、组件名）秒级搜索。
- **📂 批量管理**:
  - **全选/反选**：支持批量操作。
  - **双重删除模式**：支持仅移除数据库索引（保留文件）或 **彻底物理粉碎文件**。
  - **批量打开**：一键使用系统默认编辑器打开多个脚本。
- **🔄 自动同步**: 一键扫描 `my_pocs` 目录，自动更新数据库索引。
- **☁️ 在线更新**: 支持从 GitHub 或自定义 URL 下载最新的 POC 压缩包并自动解压。
- **📝 代码预览**: 双击列表项即可在软件内直接预览代码内容，无需切换窗口。
- **🔠 字号调节**: 支持动态调整列表字体大小，适应不同分辨率屏幕。

## 📦 目录结构

程序首次运行后会自动生成以下文件：

Plaintext

```
POC管理助手v1.0.py              # 主程序源码
poc_library.db       # SQLite 索引数据库 (自动生成)
my_pocs/             # POC 文件存放目录 (在此放入你的脚本)
```

## 📖 使用指南

1. **添加 POC**: 将你的 `.py`, `.yaml`, `.json`, `.txt`, `.md` 文件放入 `my_pocs` 文件夹（支持子目录）。
2. **建立索引**: 打开软件，点击左侧的 **"🔄 同步本地库"**。
3. **搜索**: 在顶部输入框输入关键词并回车。
4. **预览**: 双击任意搜索结果，查看文件内容。
5. **重置**: 点击 **"⚠️ 重置全库"**，你可以选择仅清空数据库记录，或者连同本地文件一起删除（慎用）。
